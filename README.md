# 🏥 AI-Powered Healthcare Prediction & Resource Management System

An intelligent healthcare web application built using **Streamlit, Python, Machine Learning, and SQLite**.  
It helps hospitals manage patients, doctors, appointments, and resources while providing AI-based disease prediction and analytics.

## Live link
https://nec-project3-aihealthcare-prediction.onrender.com

## 🚀 Features

### 🔐 Authentication
- Role-based login system (Patient, Doctor, Admin, Staff)
- Secure session management

### 🏥 Hospital Management
- Patient registration & medical history
- Doctor profiles & scheduling
- Appointment booking system
- Electronic Health Records (EHR)

### 🤖 AI Modules
- Disease prediction (Diabetes, Heart, Kidney, Cancer)
- Treatment recommendation system
- Patient outcome prediction
- Risk analysis

### 🏨 Resource Management
- Bed availability tracking
- Staff scheduling optimization
- Resource allocation system

### 📊 Dashboard & Analytics
- Interactive hospital dashboard
- Patient & doctor analytics
- Resource utilization reports

### 💬 AI Chatbot
- Symptom-based responses
- Database-driven hospital queries
- Basic medical assistance
